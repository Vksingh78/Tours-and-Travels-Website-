# 🌍 Tours and Travels Website

## 🎓 College Project

This project was developed as part of my college academic curriculum.  
The objective of this project is to design and develop a full-stack Tours and Travels web application.

---

## 📖 Project Description

The website allows users to explore travel destinations, view tour packages, and send inquiries.  
It demonstrates frontend development, backend integration, and database connectivity.

---

## 🚀 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

---

## ✨ Features

- 🏖 Browse travel destinations
- 📦 View tour packages
- 📞 Contact form
- 🔗 Backend server integration
- 🗄 MongoDB database connectivity
- 📱 Responsive design

---

## ⚙ How to Run the Project

1. Clone the repository  
git clone https://github.com/Vksingh78/Tours-and-Travels-Website.git  

2. Install dependencies  
npm install  

3. Start the server  
node app.js  

4. Open browser  
http://localhost:3000
